How to Automate Your Sponsor Reports with Google Apps Script

📅 January 15, 2026 ⏱ 8 min read ✍️ Veltrix Team
← Back to Blog

If you have even one active sponsorship, you're probably spending at least an hour every week doing the same thing: opening your spreadsheet, checking YouTube for updated stats, formatting numbers into an email, and hitting send.

Multiply that by three sponsors and you're losing half a workday every week to a task that a computer can do in 30 seconds. This guide shows you exactly how to set it up.

💡 What you'll build: A Google Apps Script that pulls your YouTube stats daily, generates a branded PDF report, and emails it to your sponsor automatically — without you touching anything.

What You Need Before Starting

  • A Google account (free)
  • A YouTube channel with at least one sponsored video
  • A YouTube Data API key (free — we'll set this up below)
  • 15–20 minutes for the initial setup
⚠️ No coding experience required. Everything in this guide is copy-paste. If you can use Google Sheets you can do this.

Step 1 — Get Your YouTube API Key

  1. 1 Go to console.cloud.google.com and sign in with your Google account.
  2. 2 Create a new project — name it anything (e.g. "Sponsor Reports").
  3. 3 Go to APIs & Services → Enable APIs and search for YouTube Data API v3. Enable it.
  4. 4 Go to Credentials → Create Credentials → API Key. Copy the key.
✅ The YouTube Data API is free for up to 10,000 requests per day. Your script will only use around 12 requests per run — one per video. You'll never hit the limit.

Step 2 — Set Up Your Google Sheet

Create a new Google Sheet. In Column A, paste the YouTube video URLs you want to track. One URL per row, starting at row 2. Row 1 is your header — label it Video URL.

Add these column headers in row 1:

A: Video URL B: Title C: Views D: Likes E: Sponsor F: Last Updated

Column E is where you put your sponsor's name for each video. This gets pulled into the PDF report automatically.

Step 3 — Add the Apps Script

In your Google Sheet, go to Extensions → Apps Script. Delete any default code and paste your script.

The script does four things when it runs:

  • Reads every URL from Column A of your sheet
  • Calls the YouTube API for each video to get live stats
  • Updates Columns B through F with fresh data
  • Generates a PDF from the data and emails it to your sponsor

Step 4 — Set the Daily Trigger

In the Apps Script editor, go to the clock icon on the left sidebar (Triggers). Click Add Trigger.

  1. 1 Set the function to runDailyReport
  2. 2 Set the event source to Time-driven
  3. 3 Set the type to Day timer
  4. 4 Set the time to 8am–9am

Click Save. That's it. Every morning at 8am, the script runs, pulls fresh stats, and emails your sponsor — automatically.

✅ From this point on you never need to think about sponsor reporting again. The bot handles it every single morning.
